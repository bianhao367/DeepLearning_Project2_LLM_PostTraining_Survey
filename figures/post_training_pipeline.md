# 后训练流程

## 概述

下图展示了完整的 LLM 后训练流程，从预训练模型到评估后的助手。

```
┌─────────────────────────────────────────────────────────────────────┐
│                    LLM 后训练流程                                     │
└─────────────────────────────────────────────────────────────────────┘

                         ┌─────────────────┐
                         │  预训练模型       │
                         │  (LLaMA, Qwen)   │
                         └────────┬────────┘
                                  │
                                  ▼
         ┌────────────────────────────────────────────┐
         │         阶段 1：微调                        │
         │                                            │
         │  ┌──────────────┐  ┌──────────────────┐   │
         │  │ 指令数据      │  │ 方法：            │   │
         │  │ (52K+ 条)     │→│ - 全量微调        │   │
         │  │ (Alpaca 风格) │  │ - LoRA           │   │
         │  └──────────────┘  │ - QLoRA          │   │
         │                    └──────────────────┘   │
         └────────────────────┬───────────────────────┘
                              │
                              ▼
         ┌────────────────────────────────────────────┐
         │      阶段 2：偏好数据收集                    │
         │                                            │
         │  ┌──────────────┐  ┌──────────────────┐   │
         │  │ 提示收集      │  │ 来源：            │   │
         │  │               │→│ - 人类排序        │   │
         │  │               │  │ - AI 反馈        │   │
         │  └──────────────┘  └──────────────────┘   │
         └────────────────────┬───────────────────────┘
                              │
                              ▼
         ┌────────────────────────────────────────────┐
         │         阶段 3：对齐                        │
         │                                            │
         │  ┌──────────────────────────────────────┐ │
         │  │ SFT → 奖励模型 → RLHF/PPO           │ │
         │  │ SFT → 偏好数据 → DPO                 │ │
         │  │ SFT → 推理 RL → GRPO                 │ │
         │  └──────────────────────────────────────┘ │
         └────────────────────┬───────────────────────┘
                              │
                              ▼
         ┌────────────────────────────────────────────┐
         │         阶段 4：评测                        │
         │                                            │
         │  ┌──────────────┐  ┌──────────────────┐   │
         │  │ 对齐后的模型  │  │ 框架：            │   │
         │  │               │→│ - lm-eval-harness│   │
         │  │               │  │ - OpenCompass    │   │
         │  └──────────────┘  │ - HELM           │   │
         │                    └──────────────────┘   │
         └────────────────────┬───────────────────────┘
                              │
                              ▼
         ┌────────────────────────────────────────────┐
         │         阶段 5：结果                        │
         │                                            │
         │  基准测试分数 + 错误分析                     │
         │  MMLU, HellaSwag, ARC, GSM8K 等           │
         └────────────────────────────────────────────┘
```

## 微调方法对比

```
┌─────────────────────────────────────────────────────────────────┐
│              微调方法对比                                         │
├─────────────┬─────────────┬─────────────┬───────────────────────┤
│   方面       │  全量微调    │    LoRA     │      QLoRA            │
├─────────────┼─────────────┼─────────────┼───────────────────────┤
│ 可训练参数   │ 100%        │ ~0.1-1%     │ ~0.1-1%               │
├─────────────┼─────────────┼─────────────┼───────────────────────┤
│ GPU 显存     │ ~60 GB      │ ~16 GB      │ ~6 GB                 │
│ (7B 模型)   │             │             │                       │
├─────────────┼─────────────┼─────────────┼───────────────────────┤
│ 训练速度     │ 基准         │ 快 1.2-1.5 倍│ 0.8-1.0 倍           │
├─────────────┼─────────────┼─────────────┼───────────────────────┤
│ 性能        │ 最佳         │ ~98%        │ ~95%                  │
├─────────────┼─────────────┼─────────────┼───────────────────────┤
│ 最低 GPU    │ A100 80GB   │ RTX 3090    │ T4 16GB               │
│             │             │ 24GB        │                       │
├─────────────┼─────────────┼─────────────┼───────────────────────┤
│ 适用场景     │ 最大性能，   │ 资源受限，   │ 非常有限的             │
│             │ 充足 GPU    │ 多任务       │ GPU 资源              │
└─────────────┴─────────────┴─────────────┴───────────────────────┘
```

┌─────────────────────────────────────────────────────────────────┐
│              对齐方法对比                                         │

## RLHF 流程详解

```
┌─────────────────────────────────────────────────────────────────┐
│                    RLHF 流程                                      │
└─────────────────────────────────────────────────────────────────┘

    ┌──────────────┐
    │   提示        │
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐      ┌──────────────┐
    │   SFT 模型   │─────→│  生成响应     │
    │   (阶段 1)   │      │              │
    └──────────────┘      └──────┬───────┘
                                 │
                                 ▼
                          ┌──────────────┐
                          │   人类排序    │
                          └──────┬───────┘
                                 │
                                 ▼
                          ┌──────────────┐
                          │   奖励模型    │──→ r(x,y)
                          │   (阶段 2)   │
                          └──────┬───────┘
                                 │
                                 ▼
                          ┌──────────────┐
                          │   PPO 训练    │──→ 优化后的策略
                          │   (阶段 3)   │
                          └──────────────┘
                                 │
                                 ▼
                          ┌──────────────┐
                          │   对齐后的    │
                          │   模型        │
                          └──────────────┘
```

## DPO 流程详解

```
┌─────────────────────────────────────────────────────────────────┐
│                    DPO 流程                                       │
└─────────────────────────────────────────────────────────────────┘

    ┌──────────────┐
    │   提示        │
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐      ┌──────────────┐
    │   SFT 模型   │─────→│  生成多个     │
    │              │      │  响应         │
    └──────────────┘      └──────┬───────┘
                                 │
                                 ▼
                          ┌──────────────┐
                          │   偏好数据    │
                          │  (优选，      │
                          │   劣选)       │
                          └──────┬───────┘
                                 │
                                 ▼
                          ┌──────────────┐
                          │   DPO 训练    │──→ 对齐后的策略
                          │  (无需 RM!)   │
                          └──────────────┘
```
